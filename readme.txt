ffmp4muxer 是一个 mp4 文件封装器

可以把 h264 + aac 封装为 mp4 视频文件

全部源代码纯手工打造，总共不超过 1000 行源代码

相比 mp4v2, ffmpeg 这样的库，ffmp4muxer 要小巧精简得多

非常适合于嵌入式 linux 的应用场合，比如 DVR、IPCAM 等产品

在网络上、在 github 上，估计你也找不到比这个更好的库了

因为我是找遍了整个互联网的开源库，没找到满意的，才自己重新打造的

如果 ffmp4muxer 帮到了你，请给个星 star 吧
